# SQL Data Analytics Project 📊

This project focuses on SQL-based Data Analytics using structured datasets.  
The goal is to practice SQL queries for business insights, exploratory analysis, and data segmentation.

## Project Overview
This project covers:

### Exploratory Data Analysis (EDA)
✔ Database Exploration  
✔ Dimensions Exploration  
✔ Date Exploration  
✔ Measures Exploration  
✔ Magnitude Analysis  
✔ Ranking Analysis  

### Advance Analytics
✔ Change Over Time Analysis  
✔ Cumulative Analysis  
✔ Performance Analysis  
✔ Part-to-Whole Analysis  
✔ Data Segmentation  

---

## Dataset Used

- Customers
- Products
- Orders

## SQL Concepts Covered
- SELECT Statements
- Aggregate Functions (`SUM`, `AVG`, `COUNT`)
- GROUP BY
- ORDER BY
- CASE WHEN
- Window Functions
- CTE (Common Table Expressions)
- Date Functions
- Joins
- Subqueries
